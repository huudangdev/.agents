# 📱 GIAO THỨC TẠO DỰ ÁN MOBILE NGUYÊN BẢN (MOBILE INIT)

> **MỤC ĐÍCH:** Thi công Mobile App bằng React Native / Flutter hoặc Swift có tích hợp Mobile Taste chuẩn.

// turbo-all

## 🟦 STAGE 1: XÁC LẬP TƯ ĐƯỜNG MOBILE DOCTRINE
*🧠 KỸ NĂNG BẮT BUỘC:* `view_file` đọc cực sâu các skills: `mobile-design-doctrine`, `mobile-developer-standards`, `mobile-ios-design`, `mobile-android-design`, `mobile-touch-animations`, `react-native-architecture` và `flutter-architecture`.
*📦 Hành Động:* Quyết định Stack (React Native + Expo hay Flutter) dựa trên độ nặng và offline của Project.

## 🟦 STAGE 2: MOBILE UI BOOTSTRAP VÀ TAILWIND FIRST
*🧠 KỸ NĂNG:* Đọc `sleek-design-mobile-apps`, `tailwindcss-mobile-first`, `ui-mobile-bootstrap` và `mobile-app-testing`.
*📦 Hành Động:* Maya tạo `MOBILE_BRAND_GUIDELINE.md` với Margin/Padding cho thiết bị cầm tay, khoảng Safe-area (iOS/Android) và cơ chế Cảm ứng lò xo (Framer/Reanimated).


## 🟦 STAGE 3: TRIỂN KHAI STATE VÀ REFACTOR VÀO AGENTS.MD
*📦 Hành Động:* Xẻ Mobile Design thành các màn hình và tạo thư mục `.brain/` cho mỗi Screen. Report trạng thái này qua file `agents.md`.
