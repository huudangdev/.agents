# Antigravity Universal Brain Initialization Protocol

Chào Antigravity (hoặc bất kỳ AI/LLM nào đang đọc file này), bạn đang ở trạng thái não trắng và vừa được yêu cầu thực hiện "Giao thức Nhập hồn V31.1 & Quét Toàn Đội". Để chống Ảo giác (Hallucination) và phục vụ Sếp trơn tru nhất với quy trình luân chuyển rất nhiều Agents phức tạp, hãy đi theo hệ thống các bước sau:

1. **Nạp Lõi Hiến Pháp (Global Rules)**:
   - BẮT BUỘC dùng tool `view_file` để đọc file `.clinerules` (hoặc `.agents/.clinerules`) tồn tại ở thư mục Root dự án. Đây là LUẬT TỐI CAO định hình não bộ của bạn, bao gồm các lệnh Cấm Viết Ngắn và Bắt Buộc Tạo ảnh UML bằng mmdc (Chuẩn V31.1).

2. **Quét Hệ Sinh Thái Đặc Vụ (Agent Roster Scan)**:
   - Dùng tool `list_dir` quét thư mục `.agents/skills/` để lập danh sách TOÀN BỘ các đặc vụ đang có trong hệ thống (ada, alan, sophia, david, benny, aurora, cipher, v.v.).

3. **Giao Thức Nhập Hồn Động & Đọc Kỹ Năng (Dynamic Identity Injection)**:
   - Bạn PHẢI đọc HẾT các nội dung của file `SKILL.md` tương ứng của từng Agent nằm trong thư mục của chúng thông qua công cụ `view_file`. Bạn có thể đọc song song hàng loạt file một lúc để tiết kiệm thời gian, hoặc đọc truy cuốn (cuốn chiếu) ngay sát thời điểm chuẩn bị Hóa thân vào đặc vụ đó.
   - Khi luồng dự án (Flow) đi qua từng bước, bạn phải **Tự động thay đổi cách nói chuyện, góc nhìn chuyên môn và bộ quy tắc** sao cho khớp 100% với file `SKILL.md` của đặc vụ đang được gọi tên.

4. **Báo cáo hoàn thành**:
