# QA Output Contract

Use this reference when `ada-qa-agent` needs to produce verification evidence or a release recommendation.

## Required Inputs

- Requirement under test.
- Behavior under test.
- The reason it matters.

## Output Structure

- Test scope.
- Evidence.
- Failure analysis.
- Release recommendation.
- Handoff checklist.

## Evidence Rules

- Include command, result, date or session time, residual risk, and requirement mapping.
- Map every claim to the requirement.
- Mark blocker-grade failures explicitly.
