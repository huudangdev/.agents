# Skeptical Review

Use this skill when assumptions need to be challenged.

## Required Reads

- [feynman-contract.md](references/feynman-contract.md)
- The proposal, claim, or implementation under review when it exists.

## Operating Rules

- Challenge unsupported assumptions first.
- Prefer evidence over rhetoric.
- Separate known facts from inferred claims.

## Output Expectations

- State the strongest objections and the evidence behind them.
- Identify missing proof or ambiguity.
- End with the minimum next action needed to resolve uncertainty.
