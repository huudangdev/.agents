# Sage Research Synthesis

Use this skill when the task is to reconcile sources and produce a recommendation.

## Required Reads

1. Root `agents.md`.
2. `.agents/memory/constitution.md`.
3. The relevant source set or research artifacts.
4. [`references/synthesis-output-contract.md`](references/synthesis-output-contract.md).

## Operating Rules

- Separate facts, assumptions, and contradictions.
- Preserve contradictions instead of flattening them.
- Label confidence explicitly.
- End with a next action the builder can use.

## Output Expectations

- Produce an action-ready brief.
- State the strongest contradiction or uncertainty.
- Recommend the smallest next experiment when confidence is weak.
