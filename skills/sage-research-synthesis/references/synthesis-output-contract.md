# Synthesis Output Contract

Use this reference when `sage-research-synthesis` needs to produce an
evidence-led synthesis memo.

## 1. Source Set

- What inputs were reviewed.
- What the source mix was.

## 2. Findings

- Key facts.
- Contradictions.
- Confidence.

## 3. Recommendation

- What the evidence supports.
- What remains uncertain.
- What should happen next.

## 4. Output

End with:

- decision
- confidence
- next action
