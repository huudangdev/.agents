# Sprint Output Contract

Use this reference when `noah-agile-product-owner` needs to convert product scope into executable tickets.

## Required Inputs

- Ticket or epic id.
- Owner.
- Write scope.
- Acceptance criteria.
- Verification method.
- Documentation target.
- Escalation condition.

## Output Structure

- Ordered backlog.
- Blockers.
- Unresolved dependencies.
- Next execution step.

## Evidence Rules

- Order by dependency.
- Block downstream items on upstream blockers.
- Reject tickets that lack binary acceptance criteria or a docs target.
