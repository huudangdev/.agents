# Mobile Touch Animations

Use this skill for motion that improves feedback, state communication, and mobile feel without harming usability.

## Required Reads

- [motion-contract.md](references/motion-contract.md)
- The current motion stack and target interaction when they exist.

## Operating Rules

- Use motion to clarify state, not to decorate.
- Preserve touchability during scale and transition states.
- Keep reduced-motion and interruptibility explicit.

## Output Expectations

- State the motion goal and the interaction it clarifies.
- Identify accessibility and performance constraints.
- Describe how the motion should be verified visually.
