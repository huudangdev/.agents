# Recon Output Contract

Use this reference when `elite6-research` needs to produce a competitive or
external intelligence brief.

## 1. Question

- What is being compared.
- Why it matters.

## 2. Sources

- Primary source types.
- Secondary sources if used.
- Source quality caveats.

## 3. Findings

- Capability differences.
- Cost or stack differences.
- Risks and limitations.

## 4. Recommendation

- What the evidence implies.
- What should be tested next.

## 5. Output

End with:

- comparison table
- key risks
- next action
