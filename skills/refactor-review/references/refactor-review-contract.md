# Refactor Review Contract

Use this reference when `refactor-review` needs to produce an audit report.

## 1. Review Scope

- Diff or files reviewed.
- Why those files matter.

## 2. Findings

- Severity.
- Evidence.
- Regression risk.

## 3. Verification Gaps

- Missing tests.
- Missing validators.
- Missing docs sync.

## 4. Recommendation

- GO
- GO WITH RESIDUAL RISK
- NO-GO

## 5. Output

End with:

- findings
- open questions
- recommendation
