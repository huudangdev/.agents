# Antigravity Brainstorming

Use this skill only when normal planning is blocked and the team needs bounded lateral options.

## Required Reads

- [brainstorm-contract.md](references/brainstorm-contract.md)
- The blocker evidence and active feature docs when they exist.

## Operating Rules

- Generate at most three options.
- Include upside, blast radius, and verification burden.
- Convert the surviving option into spec-ready input.

## Output Expectations

- State the blocker and what makes it real.
- Show the bounded options.
- End with concrete next steps that can be routed into planning.
