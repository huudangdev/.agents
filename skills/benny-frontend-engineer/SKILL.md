## 🛡️ MANDATORY PROTOCOLS (HIẾN PHÁP BẮT BUỘC V28-V32)

### Protocol 1: Bảng mã Thiết Kế Bất Di Bất Dịch (Tích hợp V4 Taste-Skill)
- **Typography:** CHỈ dùng 14px và 16px. CẤM font Serif và cấm Inter mặc định. Phải ép xài các Font: `Geist`, `Satoshi`, `Outfit`, `Cabinet Grotesk`. Heading bắt buộc phải ép cực khít (`tracking-tighter`, `leading-none`).
- **Spacing & Layout:** Hệ số khoảng cách 4px. CẤM thói quen Wrap Flex Center bản năng LLM. Ép dùng Lưới Toán học `grid-cols-12` hoặc Bento/Asymmetric layout (1 to 2 nhỏ).
- **Buttons & Radius:** Min-height 40px. Border Radius dao động 8px -> 40px tùy Context.
- **Icons & Images:** Cấm tuyệt đối chèn Emoji vào UI 🚀🔥. Phải dùng `lucide-react` hoặc `phosphor-icons` (Box 24px, stroke 1.5-2px). Yêu cầu cài thư viện trước khi code.
- **Màu Sắc (Color Palette):** Background ép hệ Zinc/Slate (`#09090b` thay vì `#000000`). Accent Color cực trầm (Saturation < 80%). CẤM glow tím chóe điện tử slop.
- **Vật lý bề mặt (Glass & Motion):** Kính nén Glassmorphism bắt buộc phải có Inner Border giả lập ánh sáng `shadow-[inset] border-white/10`. Motion ép xài Framer Motion Spring `stiffness: 100, damping: 20`, cấm ease default, cấm loading Spinner (Chỉ xài Skeleton/Shimmer chạy gradient dọc ngang).
- **Mobile Height:** Cấm dùng `h-screen`, phải dùng min-h `100dvh`.

### Protocol 2: Kỹ năng Frontend trong Môi trường Hỗ Trợ AI (V28 Agentic Design Patterns)
- **Dynamic Streaming & RAG JSON Engine:** Tích hợp Data theo Luồng (SSE). Cấm giao diện fetch tĩnh 1 cục ngồi đợi. Data trả về dưới dạng JSON Markdown Stream.
- **Render Tuyệt Đối Các UI States:** Ép buộc render các State `isThinking`, `isStreaming`, `awaitingApproval`. Dùng Shimmer Text cho trạng thái Thinking. 
- **Human-In-The-Loop Execution:** Render các Action Button cho User tương tác với Data: `onRegenerate`, `onApprove`, `onModify`. UI Component không bao giờ được thiết kế dạng Dead End.
- **Optimistic Updates:** Làm mượt thao tác Approve/Edit bằng cách Fake Success Update UI ngay tắp lự, nếu Server báo lỗi mới Rollback UI về State Red.

### Protocol 3: Kháng Ảo Giác Cơ Sở (V30.0 Awesome-Claude Matrix)
