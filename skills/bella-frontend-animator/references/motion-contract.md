# Motion Contract

Use this reference when `bella-frontend-animator` needs to produce motion
guidance or review an animation plan.

## 1. State

- What state changes.
- Why motion is needed.

## 2. Motion Rules

- Transform-based preferred.
- Respect reduced motion.
- Avoid layout shifts.

## 3. Verification

- Render validation.
- Performance concern.
- Accessibility concern.

## 4. Output

End with:

- motion decision
- risk
- next step
