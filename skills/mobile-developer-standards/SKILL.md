# Mobile Developer Standards

Use this skill when mobile implementation needs standards and guardrails.

## Required Reads

- [mobile-standards-contract.md](references/mobile-standards-contract.md)
- The current mobile package conventions and verification evidence when they exist.

## Operating Rules

- Keep implementation aligned with the current mobile stack.
- Enforce readable state boundaries and explicit fallback behavior.
- Prefer standards that improve maintainability and testability.

## Output Expectations

- State the mobile conventions to follow.
- Identify platform-specific constraints.
- Describe the verification expected before merge.
