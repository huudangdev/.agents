# QA Approval Contract

Use this reference when `eve-qa-approver` needs to produce a final QA verdict.

## 1. Scope

- Requirement under test.
- User-facing behavior.

## 2. Evidence

- Command run.
- Result.
- Requirement mapping.
- Residual risk.

## 3. Verdict

- GO.
- GO WITH RESIDUAL RISK.
- NO-GO.

## 4. Output

End with:

- findings
- verdict
- next step
