## 📦 Cách Cài Đặt (Installation)

Để sử dụng bộ Não này cho các dự án của bạn, chỉ cần tạo một Workspace tổng và clone repo này về dưới dạng thư mục `.agents`.

Mở Terminal và chạy:

```bash
# 1. Tạo một thư mục Workspace tổng cho công ty/cá nhân
mkdir marcus-workspace && cd marcus-workspace

# 2. Clone "Lõi AI" Marcus Fleet về (thư mục ẩn .agents)
git clone https://github.com/huudangdev/.agents.git

# 3. Kích hoạt Trợ lý AI của bạn (Antigravity / Cursor / OpenClaw) trong Workspace này.
```

## ⚡ Các Lệnh Tối Thượng (Slash Commands)

Hệ thống được thiết kế để tự động hóa vật lý mọi thao tác tay của lập trình viên. Bạn chỉ cần gõ lệnh vào khung chat của AI:

### 1. Khởi tạo Dự Án Chớp Nhoáng
> Gõ: `/marcus_init <Tên-Dự-Án> "<Mô tả ngắn gọn ý tưởng của bạn>"`

**Hành động của AI:**
- Tự động gõ lệnh Bash tạo thư mục `projects/<Tên-Dự-Án>`
- Tự động Copy nhân bản bộ não `.agents` và `.clinerules` đi vào dự án con.
- Đóng vai thư ký (PM Sophia) viết sẵn file Yêu cầu `PRD_RAW.md` vào dự án mới.

### 2. Kích Hoạt Ma Trận Code 10 Lớp (The Factory)
Khi đã vào thư mục dự án con (`cd projects/Ten-Du-An`), gõ:
> `/auto_software_factory`

**Hành động của AI (10 Phase Tuần Tự):**
1. **Research:** Rà soát kho dữ liệu web tìm công nghệ mới nhất.
2. **Brand Guideline & UML:** Maya tự đan Design Tokens (Luật 4px Spacing, 14/16px Typography, Radius 8-40px). Chartis tự vẽ sơ đồ Class/Sequence Diagram.
3. **Architecture:** David dựng khung Feature-Sliced Design (FSD).
4. **Debate:** Hội đồng Tòa án (Red Team) dội bom phản biện kiến trúc.
5. **DB Migration:** Tự chạy lệnh push database.
6. **Obra Superpowers Coding:** Coder phân nhỏ task 5 phút (writing-plans), áp dụng TDD (Red-Green-Refactor). Code đến đâu sinh Test đến đó. Tuyệt đối không sinh code lười nhác.
7. **Systematic Debugging:** Tự phân mảnh hai não bộ để check Lỗi Spec và Lỗi Code Quality. Gặp bug tự truy vết Root Cause nguyên thủy.
8. **QA & DevOps:** Tự động Build Docker, start server trên port, và Verify bằng cURL.
