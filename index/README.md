# Generated Context Index

This directory is generated by `.agents/scripts/build_context_index.py`.

Purpose:
- Give any model a small, stable entrypoint into project docs, skills, and code.
- Prevent "read everything" behavior that wastes tokens and causes context drift.
- Provide a lightweight architecture map/graph to guide targeted reads.

Typical usage:

```bash
python3 .agents/scripts/build_context_index.py --root .
python3 .agents/scripts/validate_context_index.py --root .
```

Outputs (generated):
- `docs_index.md`
- `skills_index.md`
- `code_index.md`
- `architecture_graph.mmd`
- `index_manifest.json`
